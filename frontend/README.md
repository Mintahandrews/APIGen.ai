# APIGen Frontend

This directory contains the Next.js app for the APIGen.ai frontend.

## Prerequisites
- Node.js 18+
- pnpm or npm

## Install
```powershell
# from repo root
cd frontend
npm install
```

## Run (dev)
```powershell
cd frontend
npm run dev
```

App runs at http://localhost:3000

## Build and Start
```powershell
cd frontend
npm run build
npm start
```
